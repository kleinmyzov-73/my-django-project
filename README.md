Tips for 2026 - Personal Blog
A modern, responsive blog application built with Django and W3.CSS. This project is designed to share career advice, technical insights, and personal development tips to help others kickstart their professional journeys in 2026.

🚀 Features
Dynamic Post Management: Full CRUD functionality via the Django Admin.

Clean UI: Minimalist design using the W3.CSS framework for a fast, responsive experience.

Sidebar Navigation: Includes "About Author" and "Popular Posts" sections for better user engagement.

Detailed Post Views: Dedicated pages for long-form content with "Back to Home" navigation.

🛠️ Tech Stack
Backend: Django (Python)

Frontend: HTML5, W3.CSS, Google Fonts

Database: SQLite

⚙️ Installation & Setup
1. Clone the repository:
>>git clone https://github.com/your-username/tips-for-2026.git
>>cd tips-for-2026

2. Set up a virtual environment:
python -m venv venv
# Windows:
venv\Scripts\activate
# Mac/Linux:
source venv/bin/activate

3. Install dependencies:
>>pip install -r requirements.txt

4.Apply migrations & start server:
>>python manage.py migrate
>>python manage.py runserver
>>python manage.py runserver
